Documentacion

Una vez en la página, vamos a poder ingresar a la parte de novedades y de funciones


Novedades

Ingresando por GET vamos a poder obtener la siguiente información:

http://localhost:3000/novedades Redirige al json con todas las novedades

http://localhost:3000/novedades/paginas Se genera el paginado

http://localhost:3000/novedades/:id Lleva al json con la novedad especifica de ese id

http://localhost:3000/novedades/buscar/nombre Vamos a poder filtrar las novedades por el nombre

http://localhost:3000/novedades/buscar/categoria Vamos a poder filtrar las novedades por la categoria


Vamos a poder hacer las siguientes acciones en base a Novedades:

Ingresando por POST

http://localhost:3000/novedades Vamos a poder crear una nueva novedad


Ingresando por PUT

http://localhost:3000/novedades/:id Vamos a poder actualizar una novedad


Ingresando por DELETE

http://localhost:3000/novedades/:id Vamos a poder eliminar una nueva novedad

Funciones

Ingresando por GET vamos a poder obtener la siguiente información:

http://localhost:3000/funciones Redirige al json con todas las funciones

http://localhost:3000/funciones/:id Lleva al json con la funcion especifica de ese id

http://localhost:3000/funciones/buscar/funcion Vamos a poder filtrar las funciones por el nombre


Vamos a poder hacer las siguientes acciones en base a Funciones:

Ingresando por POST

http://localhost:3000/funciones Vamos a poder crear una nueva funcion


Ingresando por PUT

http://localhost:3000/funciones/:id Vamos a poder actualizar una funcion


Ingresando por DELETE

http://localhost:3000/funciones/:id Vamos a poder eliminar una funcion
